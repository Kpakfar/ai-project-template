# Backlog

Scoped, queued tasks. Tasks here are ready to be picked up - they have clear acceptance criteria.

Tasks move: `proposals-ideas.md` → `backlog.md` → `current-task/task.md` → done (commit message or `_log/`).

## Format

```
### [ID] Task title
- Why: one-sentence motivation
- Acceptance criteria:
  - [ ] criterion 1
  - [ ] criterion 2
- Estimated effort: S / M / L (under 1h / half-day / multi-day)
- Dependencies: list other tasks/issues this blocks on
- Links: docs, tickets, PRs
```
