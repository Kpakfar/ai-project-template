Quick reference:

Source code:
src/                               # main application package
  api/                             # FastAPI routes
    routes/                        # route modules
    deps.py                        # FastAPI dependencies
    main.py                        # app factory
  core/                            # config, logging, exceptions
    config.py                      # pydantic-settings config
    logging.py                     # logging setup
    exceptions.py                  # custom exception classes
  domain/                          # business logic, pure functions
    models.py                      # pydantic domain models
  infra/                           # I/O: database, vector store, LLM clients
    db.py                          # Postgres connection, session management
    vectorstore.py                 # pgvector / Chroma client
    llm.py                         # LLM client wrappers
  services/                        # orchestration of domain + infra
    retrieval.py                   # RAG retrieval logic
    generation.py                  # RAG generation logic
